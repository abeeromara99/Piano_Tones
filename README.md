# Piano
-This project utilizes Atmega32 microcontroller,keypad,LCD (16x2)character and Buzzer  

# Driver Implemented:  
1-DIO driver  
2-LCD driver  
3-KEYPAD driver    
4-TIMERS driver    

# How it works ?  
-we activate fast PWM mode on Timer 0 and change frequency using Keypad and show in my LCD which tone i'm pressed
